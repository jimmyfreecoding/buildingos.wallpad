# 智能屏端 Web 应用

一个基于 Vue 3 + Vite 开发的智能楼宇管理系统屏端应用，专为智能建筑环境监控和设备控制而设计。

## 🏢 项目概述

本项目是一个智能屏端 Web 应用，主要用于智能楼宇的环境监控、设备控制和数据展示。通过 MQTT 协议实时获取传感器数据，为用户提供直观的环境信息和便捷的设备操作界面。

## ✨ 主要功能

### 🌡️ 环境监控
- **温湿度监测**：实时显示室内外温度和湿度数据
- **空气质量监控**：监测 PM2.5、CO₂、甲醛、TVOC 等空气质量指标
- **环境评估**：根据传感器数据提供环境质量评级（优/良/中等/安全等）

### 🏠 智能控制
- **空调控制**：支持温度调节、模式切换（制冷/制热/送风）
- **照明控制**：智能灯光开关和亮度调节
- **群控功能**：支持区域设备的批量控制

### 📊 数据展示
- **实时数据**：通过图表和进度条直观展示各项环境指标
- **历史趋势**：环境数据的历史变化趋势分析
- **多楼层支持**：支持不同楼层和区域的数据展示

### 🎨 用户界面
- **响应式设计**：适配不同尺寸的智能屏幕
- **动态背景**：支持图片和视频背景，增强视觉体验
- **动画效果**：使用 Animate.css 提供流畅的界面动画

## 🛠️ 技术栈

### 前端框架
- **Vue 3**：现代化的前端框架
- **Vite**：快速的构建工具
- **Vue Router**：单页面应用路由管理

### UI 组件库
- **Element Plus**：企业级 UI 组件库
- **Vant**：移动端 UI 组件库

### 数据可视化
- **ECharts**：强大的数据可视化图表库

### 通信协议
- **MQTT**：物联网消息传输协议，用于实时数据通信

### 样式处理
- **Sass**：CSS 预处理器
- **Animate.css**：CSS 动画库

### 工具库
- **Axios**：HTTP 客户端
- **Moment.js**：日期时间处理
- **Screenfull**：全屏 API 封装
- **Video.js**：视频播放器

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run dev
```
应用将在 `http://localhost:5175` 启动

### 生产环境构建
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

## ⚙️ 配置说明

### MQTT 配置
项目使用 MQTT 协议与物联网设备通信，主要配置在 `src/utils/mqtt.ts` 中：
- 支持 WebSocket 连接
- 自动重连机制
- 主题订阅管理

### 代理配置
开发环境下配置了代理，用于处理跨域请求：
```javascript
proxy: {
  '/setCleanTime': {
    target: 'http://10.205.66.7:1880/',
    changeOrigin: true,
  }
}
```

## 🔧 开发指南

### 添加新的传感器类型
1. 在 `utils/mqtt.ts` 中添加新的 MQTT 主题订阅
2. 在相应的 Vue 组件中处理新的数据类型
3. 更新 UI 组件以显示新的传感器数据

### 自定义主题样式
项目使用 Sass 进行样式管理，主要样式文件位于：
- `src/assets/main.css` - 全局样式
- `src/assets/element.css` - Element Plus 主题定制
